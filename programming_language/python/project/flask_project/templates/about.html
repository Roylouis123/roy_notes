About This Application

This is a simple Flask application created to demonstrate various Flask features:

  • Routing and URL building
  • Template rendering with Jinja2
  • Static file serving (CSS, JavaScript)
  • RESTful API endpoints
  • Request handling (GET, POST)

The application maintains a simple in-memory database of items that can be viewed and manipulated through the API.

API Documentation

Endpoints:

  • GET /api/items - Get all items
  • GET /api/items/{id} - Get a specific item by ID
  • POST /api/items - Create a new item

Example API Usage:

// Get all items
fetch('/api/items')
    .then(response => response.json())
    .then(data => console.log(data));

// Get item with ID 1
fetch('/api/items/1')
    .then(response => response.json())
    .then(data => console.log(data));

// Create a new item
fetch('/api/items', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        name: 'New Item',
        description: 'This is a new item created via API'
    })
})
.then(response => response.json())
.then(data => console.log(data));
